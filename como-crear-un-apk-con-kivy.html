Como crear un apk con Kivy

Posted on vie 29 junio 2018 in programacion

Generar una aplicación Android a partir de una aplicación kivy es bastante sencillo.

Tienes que hacer lo siguiente:

  1. Descargar VirtualBox
  2. Descargar la imagen de la VM de Buildozer. Ahí tendrás un Linux preparado para generar los apk.
  3. Importar la imagen a VirtualBox
  4. Crear una carpeta en el HOST y compartirla con la máquina virtual. Especificar que se monte automáticamente.
  5. Ya en la máquina virtual, y logueado con kivy/kivy ir a la carpeta compartida donde estará tu código kivy/kivy
  6. Ejecutar:
buildozer init
  1. Ejecutar (la primera vez tardará mucho porque se descargará el SDK de Android y todo lo necesario).
buildozer -v android debug logcat
  1. El apk estará en la carpeta bin.

Ya solo te falta llevártelo a tu dispositivo móvil e instalarlo.

python kivy
